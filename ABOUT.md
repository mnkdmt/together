# Together 💛

**Личное приложение для пары: идеи → планы → воспоминания.**
Решает две боли: «не знаю, чем заняться на выходных» и «ссылки на идеи свиданий теряются в мессенджере, и непонятно, что из них она реально хочет».

- **Mini App:** https://together-six-brown.vercel.app (открывается внутри Telegram)
- **Бот:** [@togethera_bot](https://t.me/togethera_bot)

---

## Как устроено

Приложение обходит асимметрию вовлечения в паре:

- **Она остаётся в Telegram.** Пересылает боту ссылку на идею (афиша, фильм, ресторан) — бот сам вытаскивает постер, название, дату, место и спрашивает одной кнопкой, насколько хочется: 🔥 очень хочу / 🔖 интересно / 🌙 когда-нибудь.
- **Он открывает Mini App.** Видит её идеи красивыми карточками, решает «что делаем», планирует дату, а после — отмечает «мы сходили» с фото.

Каждая пара видит только своё: личность подтверждается через Telegram, все данные изолированы по парам.

## Жизненный цикл идеи

**Идея** (из бота, из афиши или руками) → **План** (выбрана дата, попадает в календарь) → **Было** (память пары с фото и настроением).

## Функционал

### 🏠 Сегодня
Главный экран-решение: «Чем займёмся сегодня?». Наверху — топ-желание (карточка с постером и кнопкой «Запланировать»), ниже — идеи послабее и ближайшие запланированные свидания со счётчиком дней. Фильтр «Все / Выезд / Дома».

### 🎬 Афиша
Встроенный каталог событий города (Москва, источник KudaGo) с рекомендациями:
- **Рекомендации** — события отсортированы под вкус пары, на карточках бейдж «% вам». Вкус учитывает выбранные категории и жанры, историю лайков, планов и походов, близость даты.
- **Опрос вкусов** — плитки категорий (концерты, кино, спектакли, выставки, вечеринки, квесты, экскурсии, актив), внутри каждой — свои жанры (рок/джаз/стендап, комедия/ужасы, живопись/фото…). Настраивается с экрана Афиши (иконка-ползунки) или из Профиля.
- **❤️ Лайк** — событие падает в «Понравившееся» и в общий список идей, дальше живёт обычным циклом.
- **Понравившееся** и **Где были** — фильтры по своим данным.

### ➕ Создание (ссылка-first)
Одно поле + кнопка «Вставить» из буфера:
- **Ссылка** → авто-извлечение: постер, название, дата, время, место, категория (угадывается). Работает почти с любым сайтом: афиши, кинопоиск, timepad, обычные сайты без OG-разметки, заблокированные для дата-центров сайты (через рендер-fallback). Сайты за капчей дают чистый заголовок без мусора.
- **Текст** → быстрая идея без ссылки.
- Детали (выезд/дома, категория, заметка) свёрнуты — заполнять не обязательно.
- Второй режим «Покупки» — список покупок (см. ниже).

### 📋 Список
Разбор входящих идей: сортировка по интенсивности, свайп вправо — в план, влево — «когда-нибудь», фильтры.

### 📅 План
Календарь запланированных свиданий. Дату можно менять тапом по плашке в детали.

### 🗂 Деталь идеи
Постер (или стильная заглушка с иконкой категории), интенсивность, кто добавил, цитата-заметка, «Когда/Где/Цена», **Яндекс-карта места** и кнопка «Построить маршрут», ссылка на источник, **🎟 билет** (см. ниже), «Запланировать» / «Мы сходили» / «Поделиться с Аней».

### 🎟 Билеты
К любой идее можно прикрепить PDF-билет или скрин — хранится приватно, открывается в один тап в день свидания. Билет едет с идеей через весь цикл.

### 💛 Было
Память пары: сетка состоявшихся свиданий с **реальными фото**. Фото добавляется в момент «Мы сходили» (со сжатием на устройстве) или позже — из детали воспоминания («Добавить/Заменить фото»). Плюс настроение (Огонь/Хорошо/Так себе) и кнопка «Повторить свидание».

### 🛒 Покупки
Общий список пары: он добавляет из аппа, она — текстом боту («молоко, хлеб» → бот разложит и спросит тему: еда/маркетплейсы/другое). Когда он отмечает её позицию купленной — **бот сам пишет ей** «💛 Дима купил: …».

### 👤 Профиль
- **Пара:** название, «Пригласить партнёра» (deep-link, партнёр входит в ту же пару), «Выйти из пары», «Поделиться с другом» (у друга создаётся своя отдельная пара).
- **Важные даты:** начало отношений (живой счётчик «вместе N дней»), свадьба, дни рождения обоих + отсчёт до ближайшего события.
- **Вкусы для рекомендаций** — вход в опрос.
- **Статистика** (только владелец): пользователи/пары.

### 🤖 Бот @togethera_bot
- Ссылка → идея с постером/датой/местом + кнопки интенсивности.
- Текст → покупки с выбором темы.
- `/start` — приветствие с баннером, кнопка «Поделиться с друзьями», меню-кнопка открывает Mini App.
- Инвайт-ссылки (`?start=inv_…`) — присоединение партнёра к паре.
- Уведомления о выполненных покупках.
- Открыт всем: любой новый человек автоматически получает свою изолированную пару.

## Дизайн

Тёмный **Apple Liquid Glass**: глубокий aubergine-чёрный фон, акцент ember-orange, стекло (blur) только на слое управления, контент-карточки плотные с реальными постерами, заголовки — контрастный сериф (Instrument Serif). Брендовый загрузочный экран с пульсирующим пламенем.

## Технологии

- **Фронт:** self-contained PWA (один HTML, без сборки) + Telegram WebApp SDK; внутри Telegram — Mini App, снаружи — обычный PWA с демо-данными.
- **Бэкенд:** Vercel serverless (webhook бота, data-proxy, извлечение ссылок, афиша, загрузка файлов), Supabase (Postgres + приватный Storage).
- **Безопасность:** личность = подписанный Telegram initData, проверяется на сервере; все операции скоуплены по паре; анонимный доступ к БД закрыт (RLS); фото и билеты — в приватных бакетах с временными подписанными ссылками.
